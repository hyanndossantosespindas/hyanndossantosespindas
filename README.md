### 🗣️ VAOX — Voz Análise Organizacional X
> Plataforma SaaS de NLP para análise de reuniões comerciais. Detecta risco de churn, menções de concorrentes e oportunidades de upsell. Desenvolvido em equipe de 5 pessoas e apresentado à banca da **TOTVS**.

### 🛰️ PyroSat — Monitoramento de Incêndios Florestais
> Sistema de detecção e resposta a incêndios com integração de dados dos satélites GOES-16, Sentinel-2 e VIIRS. Usa algoritmos BFS/Dijkstra para roteamento e banco Oracle para persistência. Inclui modelagem 3D em AR/VR (Autodesk Maya).

`Python` `Java DDD` `Oracle SQL` `Arduino` `Autodesk Maya`
